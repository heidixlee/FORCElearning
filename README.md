# FORCElearning
A simple implementation of FORCE learning teaching a random, chaotic, sparsely connected neural network to learn how to reproduce a simple summation of 4 sine waves.

force_result.png showcases the results, comparing the target function with the network's output.

The error of the network is 0.01 with this specific set of randomized numbers and parameters.

Based on the paper "Generating Coherent Patterns of Activity From Chaotic Neural Networks" by Sussilo & Abbott (2009)
